# Deployment

- Repository URL: https://github.com/prash/content-intelligence-dashboard
- Live URL: Pending GitHub repository creation and publishing
- Deployment date: 2026-07-13
- Redeploy steps:
  1. Create or confirm the GitHub repository named content-intelligence-dashboard.
  2. Push the project with `git push -u origin main`.
  3. Enable GitHub Pages from the repository settings and publish the root folder.
  4. Open the resulting GitHub Pages URL.
- Future update steps:
  1. Edit the dashboard files in `public/`.
  2. Commit the changes with `git add .` and `git commit -m "Update dashboard"`.
  3. Push the updates with `git push origin main`.
